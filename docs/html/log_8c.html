void print_level_prefix ( log_level_t level)

Print the colored log level to stdout/stderr.

This is an internal function. It prints the log header (e.g. [INFO]) to stdout (or stderr, if the log level is LOG_ERROR.) These headers are colored using ANSI escape codes

See also
src/utils/log/ansi_escape_code_def.h for the color definitions.

void write_log_with_tag ( log_level_t level,
const char * tag,
const char * title,
const char * content,
... )

Print the log tag, title and content to stdout/stderr.

This function outputs the log tag, title and message with the approate header. e.g. [INFO] Main - Greetings: Hello.

In the example above, "Main" is the tag, "Greetings" is the title and "Hello." is the message body.

If the log level is LOG_ERROR, the message will be printed to stderr. Otherwise the log is printed through stdout.
