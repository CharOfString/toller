\doxysubsection{Detailed Description}
Struct to hold the response of path query functions. 

The struct contains a status code to indicate success or failure, with a {\ttfamily status\+\_\+code} of {\ttfamily 0} indicating succeed and {\ttfamily -\/1} indicating fallback/downgraded performance, and {\ttfamily -\/2} indicating failure. The {\ttfamily path} member holds the resulting path if found.
