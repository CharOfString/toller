Get the system data directory path. 

The function first gets {\ttfamily XDG\+\_\+\+DATA\+\_\+\+HOME} variable, and returns if it is set. If not, it will try to get {\ttfamily HOME} variable and return the path as {\ttfamily HOME/}.local/share if avaliable. If failed, then it will return {\ttfamily NULL} in the path part. \begin{DoxyNote}{Note}
The function is actually returning a struct, not just a string.
